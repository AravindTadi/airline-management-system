## Steps to start the application:
1. Clone the repository
```
git clone https://github.com/gopinathsjsu/team-project-202project_team6.git
```
2. Go to Backend Repository
```
npm install
npm start
```
3. Go to Frontend Repository
```
npm install
npm start
npm run build
```
